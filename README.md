# U-BOOT

These `u-boot` binaries are intended for use with `Armbian`, `OpenWrt`, and `FnNAS` systems, such as [amlogic-s9xxx-armbian](https://github.com/ophub/amlogic-s9xxx-armbian), [amlogic-s9xxx-openwrt](https://github.com/ophub/amlogic-s9xxx-openwrt), and [fnnas](https://github.com/ophub/fnnas). They are automatically downloaded and integrated during the system build process.

这些 `u-boot` 引导文件适用于 `Armbian`、`OpenWrt` 和 `FnNAS` 系统，例如 [amlogic-s9xxx-armbian](https://github.com/ophub/amlogic-s9xxx-armbian)、[amlogic-s9xxx-openwrt](https://github.com/ophub/amlogic-s9xxx-openwrt) 和 [fnnas](https://github.com/ophub/fnnas)。在构建相关系统时，它们会被自动下载并集成。
